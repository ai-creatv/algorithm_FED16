# 과정 리뷰

## 과정 기초

- 코딩테스트에서 입출력 다루기
- 알고리즘의 시간/공간 복잡도
- JavaScript 기본 문법/클래스 구현

## 자료구조

- 추상자료형 (ADT)
- 배열 (Array)
- 리스트
  - 배열 리스트 (Array List)
  - 연결 리스트 (Linked List)
- 스택 (Stack)
- 큐 (Queue)
- 트리 구조
  - 이진 트리 (Binary Tree)
- 그래프 구조
  - DFS, BFS

## 알고리즘

- 재귀 함수
- 문자열 다루기
- 해쉬 테이블 (`Map` class)
- 동적 계획법 (Dynamic Programming; DP)
- 검색 트리 (Re**trie**val Tree; Trie)
